Introducing

This is a Machine learning deployment example: This AI was trained to calculate anyone's chances of surviving the Titanic Disaster.

You can type some hypothetical information such as your age, gender, city and many more to calculate your chances of surviving.

Please be aware "this is not a game" but a educational and graphical way to better understand how AI is deployed, and We must respect the memories of those whose died during this disaster.

About this AI

This AI was trained using the Kaggle Titanic Competition data set.

It was built using CI/CD methodology, and End to End Machine Learning Cycle. It was trained to learn patterns from the people in the titanic such their age, gender, number of relatives that were traveling with them and many more.

In a nutshell: This AI gives a numerical importance to each of these elements (Sex, Age, Cabin, Class) known as weights. Then it will compare the information that you might provide with people who had things in common with you and, finally, label your results as survived / no survived

The AI training process is far more complicated, and if you are interesting in knowing how it works, You can visit my Post in Kaggle to learn how to train this kind of AI, or Check my GitHub Repository to learn some Deployment Techniques
